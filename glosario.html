Glosario

IoT (Internet de las cosas)
Conexión de objetos físicos a internet.
Sensor:
Dispositivo que detecta y mide información del entorno
Actuador:
Componente que ejecuta acciones según señales recibidas
Red:.
Conjunto de dispositivos interconectados para compartir datos.
Big Data:
Análisis de grandes cantidades de datos generados por dispositivos IoT.
Smart home:
Casa equipada con dispositivos IoT automatizados.
API:
Interfaz que permite la comunicación entre aplicaciones.
Ciberseguridad: (IoT)
Protección de datos y sistemas conectados.
Domótica:
Automatización de funciones en el hogar.
Interoperabilidad:
Capacidad de distintos dispositivos para funcionar juntos.
